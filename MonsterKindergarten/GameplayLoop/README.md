# GameplayLoop

클릭으로 재화를 얻고, 드래그한 같은 등급 슬라임을 합쳐 상위 등급을 해금하며, 저장된 상태와 업그레이드를 스폰 시스템에 반영합니다.

## 파일

| 파일 | 역할 |
|---|---|
| [`Clicker.cs`](./Clicker.cs) | 누른 시간과 이동 거리로 클릭·드래그를 구분합니다. |
| [`PointCalculator.cs`](./PointCalculator.cs) | 수동·자동 클릭별 고정·비율 업그레이드를 조합해 획득량을 계산합니다. |
| [`MergeManager.cs`](./MergeManager.cs) | 같은 등급 여부를 검증하고 상위 등급 변경, 상태 저장, 풀 반환을 조정합니다. |
| [`SpawnManager.cs`](./SpawnManager.cs) | 저장된 활성 슬라임 복원, 자동 생성과 스폰 관련 업그레이드를 적용합니다. |

## 게임 루프

```text
슬라임 클릭 → 재화 획득 → 업그레이드 구매
                         ↓
슬라임 자동 생성 ← 생성 간격·최대 개수 강화
        ↓
같은 등급 드래그 머지 → 상위 등급 해금 → 더 높은 획득량
```

## 설계 포인트

- 클릭과 드래그를 동일 입력에서 명확히 구분합니다.
- 기본 획득량, 고정 보너스, 비율 보너스 계산을 UI나 슬라임 클래스에서 분리합니다.
- 머지 시 런타임 오브젝트 변경과 저장 도메인 변경을 하나의 흐름에서 일관되게 처리합니다.
