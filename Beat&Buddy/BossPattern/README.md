# BossPattern

현재 화면에 활성화된 노트를 거리·종류·진행도 기준으로 조회하여 보스 패턴이 기존 리듬 오브젝트를 선택적으로 변형할 수 있게 합니다.

## 파일

| 파일 | 역할 |
|---|---|
| `NoteController.cs` | 활성 노트 목록을 조건별로 필터링하고 무작위 또는 우선순위 기반 결과를 반환합니다. |

## 활용 예

- 판정 지점에 가장 가까운 노트 선택
- 특정 방향 노트만 선택
- 특정 진행도 범위의 노트만 선택
- 앞으로 등장할 노트 중 일부를 무작위 선택

보스 패턴의 개별 연출 코드는 제외하고, 패턴들이 공유하는 노트 조회 계층만 포함했습니다.
