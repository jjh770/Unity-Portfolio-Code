# RhythmCore

음악 재생, 노트 생성, Beat 기반 이동과 입력 판정을 동일한 DSP 시간축으로 연결합니다.

## 파일

| 파일 | 역할 |
|---|---|
| `SongPlayManager.cs` | 예약 재생 시각과 현재 음악 시간을 DSP 클럭으로 계산합니다. |
| `NoteSpawner.cs` | 목표 Beat보다 일정 구간 앞서 노트를 풀에서 생성하고 활성 목록을 관리합니다. |
| `Note.cs` | 현재 Beat를 기준으로 노트 위치를 계산하고 Hit·Miss 상태와 반환을 처리합니다. |
| `JudgeManager.cs` | 입력 방향과 시간 오차가 가장 작은 노트를 선택해 Perfect·Good·Bad·Miss를 판정합니다. |

## 시간 흐름

```text
DSP 예약 재생 시각
→ 현재 음악 시간
→ 현재 Beat
→ 노트 진행도와 위치
→ 입력 시 목표 Beat와의 오차
→ 판정 결과
```

## 설계 포인트

- 노트는 매 프레임 속도를 누적하지 않고 현재 음악 Beat에서 위치를 다시 계산합니다.
- 한 입력에 여러 노트가 후보가 되면 시간 오차가 가장 작은 노트 하나만 선택합니다.
- 생성과 반환은 풀을 사용하며, 재사용 시 Tween과 판정 상태를 초기화합니다.
