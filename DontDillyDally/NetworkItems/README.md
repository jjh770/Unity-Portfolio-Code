# NetworkItems

PhotonView 소유권, 아이템 보유 상태, 컨테이너 적재 상태와 오브젝트 풀 재사용을 함께 다루는 네트워크 아이템 코드입니다.

## 파일

| 파일 | 역할 |
|---|---|
| [`NetworkItemOwnership.cs`](./NetworkItemOwnership.cs) | 소유권 요청·반환·타임아웃·콜백과 동시 획득 경쟁 상태를 관리합니다. |
| [`HoldableItemNetworkSync.cs`](./HoldableItemNetworkSync.cs) | 위치·회전뿐 아니라 보유자와 컨테이너 적재 상태를 커스텀 직렬화합니다. |
| [`ItemRecycleUtility.cs`](./ItemRecycleUtility.cs) | 마스터가 네트워크 오브젝트를 안전하게 회수할 수 있도록 소유권 확보 후 제거합니다. |

## 핵심 흐름

```text
플레이어 상호작용 요청
→ 로컬 소유 여부 확인
→ 필요하면 Photon 소유권 요청
→ 마스터가 현재 보유자·승인된 요청자를 검증
→ 이전 성공 시 콜백 실행
→ 보유 상태와 위치를 다른 클라이언트에 직렬화
```

## 설계 포인트

- PhotonView 소유권 이전과 보유 상태 전파 사이의 짧은 시간차를 별도 승인 상태로 방어합니다.
- 컨테이너에 들어간 아이템은 불필요한 월드 위치 전송을 생략합니다.
- 컨테이너에서 나온 첫 패킷과 풀 재사용 첫 패킷은 보간하지 않고 즉시 적용해 잘못된 이동 궤적을 방지합니다.
- 네트워크 오브젝트 제거 권한이 없으면 먼저 마스터 소유권을 확보한 뒤 회수합니다.
